API DOCUMENTATION

getBotID

Get the ID of a bot in our database. Url: /getBotID.php GET request Parameters: username - username of the bot Ex: /getBotID.php?username=@vote

getUserVote

Get the vote that a user gave to your bot. Url: /getUserVote.php GET request Parameters: user_id - Telegram user ID, bot_Id - the id of the bot in our database. Ex: /getUserVote.php?bot_id=1&user_id=141691961
